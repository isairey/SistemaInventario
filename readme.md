## 📦 Descripción

Sistema de gestión de inventario diseñado para administrar productos, existencias y movimientos de almacén de manera eficiente.  
Permite llevar un control preciso de entradas, salidas y disponibilidad de productos en tiempo real.

---

## 🚀 Características Principales

### 📋 Gestión de Productos
- Registro de productos  
- Clasificación por categorías  
- Edición y eliminación  
- Control de stock  

---

### 📦 Control de Inventario
- Entradas y salidas de productos  
- Actualización automática de existencias  
- Alertas de bajo inventario  

---

### 📊 Reportes
- Reportes de inventario  
- Historial de movimientos  
- Análisis de stock  

---

### 👥 Gestión de Usuarios
- Administración de usuarios  
- Control de accesos  
- Roles (admin / usuario)  

---

## 🛠️ Tecnologías Utilizadas

- HTML  
- JavaScript  
- Blade (Laravel)  
- PHP / Hack  
- T-SQL
